  # 🚨 Fraud Detection Dashboard

An AI-powered Fraud Detection Dashboard built using **Streamlit**, **LightGBM**, and **SHAP Explainability**.  
This project helps analyze transactions, detect fraudulent activities, and visualize model explanations interactively.

---

# 📌 Features

✅ Real-time Fraud Detection  
✅ Fraud Probability Prediction  
✅ Interactive Transaction Explorer  
✅ SHAP Explainability  
✅ Feature Importance Visualization  
✅ Fraud Analytics Dashboard  
✅ Machine Learning using LightGBM  
✅ Streamlit Web App  

---

# 📊 Dataset

This project uses a sampled version of the IEEE-CIS Fraud Detection dataset for lightweight deployment on Streamlit Community Cloud.

### Important Columns

- TransactionID
- TransactionAmt
- isFraud

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/Fraud-Dashboard.git
```

## Move into Project Directory

```bash
cd Fraud-Dashboard
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Locally

```bash
streamlit run app.py
```

---

# ☁️ Streamlit Cloud Deployment

Deploy directly using Streamlit Community Cloud:

https://share.streamlit.io

## Steps

1. Push project to GitHub
2. Login to Streamlit Cloud
3. Click **New App**
4. Select repository
5. Choose `app.py`
6. Click **Deploy**
